# Word Collector Pro | 智能词霸 🚀

一款为您量身定制的沉浸式英语学习辅助插件。它能帮您在流畅阅读的同时，毫不费力地积累词汇与地道表达。

---

## ✨ 核心亮点

### 1. 沉浸式划词体验 (Smart Lookup)
- **极简弹窗**：全新设计的弹窗界面，中英对照，排版舒适。
- **❤️ 心动收藏**：无需繁琐点击，点击右上角心形图标即可一键收入词库。
- **自动语境捕捉**：收集单词时自动保存当前句子，助您在回温时关联真实语境。
- **精准翻译**：调用有道与自由字典 API，支持音标、语音朗读及词性解析。

### 2. 独家“精选句库” (Sentences Management)
- **独立管理**：将单项词汇与长难句分开存储，逻辑更清晰。
- **卡片式布局**：长句通过阅读优化过的卡片进行展示，美观且易读。
- **一键复用**：支持一键复制原文与语音朗读，方便后续写作或背诵。

### 3. 数据看板与趣味学习
- **🆕 学习热力图 (Heatmap)**：在管理页点亮您的“贡献墙”，可视化每日积累进度。
- **🎮 趣味测验 (Quiz Game)**：内置四选一挑战模式。答对赚积分，答错有反馈，停不下来的“升级”快感。
- **沉浸闪卡**：经典的 Flashcards 模式，深度测试主动回忆能力。

### 4. 数据完全掌控
- **一键备份/导入**：支持 JSON 格式的一键导出与智能合并导入。
- **CSV 导出**：完美兼容 Excel 或导入 Anki (支持 Context 语境列)。

## 🛠 安装与使用

### 安装步骤
1. 下载本项目或 Clone 到本地。
2. 打开 Chrome 浏览器，进入 `chrome://extensions/`。
3. 开启右上角的 **“开发者模式”**。
4. 点击 **“加载已解压的扩展程序”**，选择本项目文件夹。

### 使用技巧
1. **收集**：在网页中选中不认识的词，点击弹出的心形图标。
2. **管理**：点击插件图标 -> 选择“我的库 / Library”或“精选句子 / Sentences”。
3. **复习**：点击“复习模式 / Study”开启闯关或闪卡。
